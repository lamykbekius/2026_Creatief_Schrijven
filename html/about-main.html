    About This Website

    This is a sample website to showcase how Keystroke Loxensis Light can be used to host visualisations of keystroke-logged writing processes. The visualisations are specifically designed to facilitate text genetic analysis or to be used in creative writing courses.

    Dataset and Reusability

    You can use Keystroke Loxensis Light to visualise your own keystroke-logged writing processes encoded in TEI-XML.

    1. Encode the keystroke logging data in TEI-XML
      • Manually: use the Encoding Manual for Keystroke Logging Data to make sure you create a valid encoding.
      • (Semi-)Automatically: use Keystroke Procera (in development) to automatically generate a baselayer TEI-XML encoding.
    2. Replace the existing TEI-XML files (in /xml) with your own by following the existing folder structure (/xml/authorLastname) and file naming (authorLastname_01_s01.xml).
